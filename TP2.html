Le Format HTML - Feuille de style CSS

Présentation du TP2 :

Cet exercice permet de comprendre la mise en oeuvre de feuilles de styles CSS.

Modifie le fichier TP2.html avec notepad++ pour qu'il pointe le fichier CSS style-tp2.css et enregistre le fichier modifié.
Rafraîchis l'affichage du navigateur avec F5 ou ctrl-F5 et observe les changements.
Modifie la feuille de style style-tp2.css avec notepad++ pour obtenir le rendu souhaité dans le navigateur.

Rappel des points importants du rendu :

  • Titre sur la page WEB : centré, blue, police Arial, taille 4em
  • Titres de chapitres : gras, violet, police Comic sans MS, taille 1.2em
  • Tout le texte normal : noir, police Arial, taille 1em
  • Mots clés : gras, italique, blueviolet
  • Liste : classe liste1, cercle vide en début de ligne
  • Fond de la page WEB : lightgray
  • L'image du logo du lycée est à gauche de la page

Un exemple de liste avec des mots clé extrait du mode d'emploi du logiciel NVU :

  • La commande Editeur CSS du menu Outils permet d'editer et de compléter la feuille de style de cette page.
  • Le bouton Regle permet d'ajouter une entrée dans la feuille de style.

Conclusion :

Ce second TP avait pour objectif de te faire comprendre le principe des feuilles de styles CSS.
